# Fight Forged Media Marketing Site

Single-page marketing site for Fight Forged Media built with Next.js 14, TypeScript, and Tailwind CSS.

## Local development

1. Run `npm install`
2. Run `npm run dev`
3. Open [http://localhost:3000](http://localhost:3000)

## Deploying to Vercel

1. Push this project to GitHub
2. Import the repo into Vercel
3. Vercel will detect Next.js automatically
4. Deploy with the default build settings
